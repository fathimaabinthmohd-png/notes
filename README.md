# Travel App

A simple Flask web application for managing travel notes.

## Features

- View a list of notes on the home page
- Add new notes via a form
- Clear all notes

## Live Site

The app is deployed at: https://travel-app-avga.onrender.com

## Running Locally

1. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

2. Run the application:
   ```
   python app.py
   ```

3. Open your browser to http://localhost:5000

## Deployment

This app is configured for deployment on Render using the `render.yaml` file.
